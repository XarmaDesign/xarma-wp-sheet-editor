== Description ==
Un editor dinamico in stile foglio di calcolo per gestire contenuti WordPress (post, pagine, CPT) con aggiornamento in tempo reale via Ajax.

== Features ==
* Modifica titoli, status, date e colore personalizzato
* Filtro testo in tempo reale
* Supporto CPT e WPML/Polylang
* Interfaccia moderna e minimale

== Installation ==
1. Carica la cartella `xarma-wp-sheet-editor` nella directory `/wp-content/plugins/`
2. Attiva il plugin da Plugins nel pannello WP
3. Vai su “WP Sheet Editor” nel menu admin
