== Description ==
Editor a foglio di calcolo per WordPress: modifica contenuti, CPT e campi personalizzati in stile Excel. Supporta WPML, autosave, backup, drag & drop.

== Features ==
* Modifica in linea via AJAX
* Clona / cestina / crea post
* Supporto WPML e Polylang
* Backup automatici e rollback
* Drag & drop ordine post
* Esporta Excel (.xlsx)

== Installation ==
1. Carica la cartella `xarma-wp-sheet-editor` in `/wp-content/plugins/`
2. Attiva il plugin da "Plugin > Plugin installati"
3. Vai su "Xarma Sheet" nel pannello admin
